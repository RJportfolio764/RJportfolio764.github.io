How I used AI to build the project

This project started as a learning experiment: could a small AI-agent workflow turn a dataset description into a polished Power BI dashboard and public case study faster than a manual build? The result became the first proof of concept for the dashboard pipeline.

Key Insights
The dashboard story behind the AI jobs market.
Insight 01

The dashboard makes compensation visible first

The case study leads with salary views so viewers can quickly compare which AI job categories appear most valuable in the market.

Insight 02

Demand is easier to understand when roles and time are shown together

The dashboard pairs role/category bars with a monthly demand-score trend, making it easier to see both where jobs cluster and how demand changes over time.

Insight 03

Experience level changes the story

The experience-level view separates lead, senior, mid, and entry roles so the market is not flattened into one average salary number.
